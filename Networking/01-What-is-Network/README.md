## What is a Network?

A Network is a collection of two or more devices connected together to share data, resources and services.

## Why is Network Important?
Data Sharing

Resource sharing

Internet access

Communication

## Types of Networks

# LAN(Local Area Network)

A LAN is a Network that covers a small geographical area.

# Example
 Home wi-fi
 
 office network

# MAN(Metropolitan  area network )

Covers a city or campus.

# WAN(Wide Area Network)

 A WAN Covers a large geographical area,often connecting multiple LANs.

 # Example
 Internet

## Real world Example
When your laptop connects to your home Wi-Fi and accesses the internet, it is using a network.

## Key Points
 Devices communicate using IP addresses.

 Data is transferred in the form of packets.

 Routers and switches help devices communicate.

## Bandwidth vs Latency

## Bandwidth

Bandwidth is the maximum amount of data that can be transferred over a network in a given time.

Unit

bps

Kbps

Mbps

Gbps

# Example

100 Mbps connection means up to 100 Megabits per second can be transmitted.

# Think of it as:

"Bandwidth = Width of a highway"

A wider highway allows more cars to travel simultaneously.

# Latency

Latency is the time taken for data to travel from source to destination.

Unit

Milliseconds (ms)
# Example

5 ms → Excellent

30 ms → Good

100 ms → Noticeable delay

300 ms → Poor

# Think of it as:

Latency = Time taken by the first car to reach the destination



## Interview Questions
# Q1. What is a network?
Answer: A network is a group of interconnected devices that communicate and share resources.

# Q2. What are the main types of networks?
Answer: LAN, MAN, and WAN.

# Q3. What is the difference between bandwidth and latency?
Bandwidth is the maximum amount of data that can be transferred per second, while latency is the delay before data reaches its destination.

# Q4. Which is more important for online gaming?
Low latency, because quick response times matter more than high data transfer rates.

# Q5. Can you have high bandwidth and high latency at the same time?
Yes. For example, a satellite internet connection can offer high bandwidth but still have high latency due to the long distance signals must travel.

# Q6. Is download speed the same as bandwidth?
Not exactly. Bandwidth is the maximum capacity of the connection, while actual download speed also depends on factors like network congestion, server performance, and latency.
